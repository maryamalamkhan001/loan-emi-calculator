# 💰 Professional Loan EMI Calculator

A professional Loan EMI Calculator developed using Python and Streamlit. This web application helps users calculate monthly EMI, total payment, and total interest for any loan amount quickly and accurately.

## 📌 Features

* Calculate Monthly EMI (Equated Monthly Installment)
* Calculate Total Loan Repayment Amount
* Calculate Total Interest Payable
* User-Friendly Interface
* Fast and Accurate Financial Calculations
* Responsive Web Application

## 🚀 Technologies Used

* Python
* Streamlit

## 📊 How It Works

The user enters:

* Loan Amount
* Annual Interest Rate (%)
* Loan Tenure (Years)

The application then calculates:

* Monthly EMI
* Total Payment
* Total Interest
* Loan Summary

using the standard EMI formula.

## 📝 EMI Formula

```text
EMI = [P × R × (1 + R)^N] / [(1 + R)^N – 1]
```

Where:

* P = Loan Amount
* R = Monthly Interest Rate
* N = Number of Monthly Installments
